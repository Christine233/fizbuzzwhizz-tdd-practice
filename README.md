# fizbuzzwhizz-tdd-practice

# Tasking
```
fizBuzzWhizz: 返回0-100对应的正确单词序列
输入：
a: Number
b: Number
c: Number
输出：
words: String

matchLiteral: 按数字字符匹配第一个特殊数（规则5）
输入：
num: Number
table: [{
  number: Number,
  word: String
}]
输出：
word: String

getModCollection: 得到某数字的特殊数因子的集合（规则3和4）
输入：
num: Number
table: [{
  number: Number,
  word: String
}]
输出：
modCollection: [{
  number: Number,
  word: String
}]

generateOutput: 将某因子集合转化成对应单词
输入：
modCollection
输出：
words: String
```
